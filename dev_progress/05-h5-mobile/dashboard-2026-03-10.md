## 🎨 设计特点

### 1. 视觉设计

**配色方案**:
- 背景：深蓝渐变 `#0c1929 → #1a2a4a`
- 主色：青色 `#00f5ff`（标题动画）
- 状态色：
  - 作业中：绿色 `#52c41a`
  - 待命：蓝色 `#1890ff`
  - 维保：橙色 `#faad14`
  - 故障：红色 `#ff4d4f`

**动画效果**:
- 标题渐变动画（3 秒循环）
- 卡片发光效果
- 标记点悬停放大

### 2. 响应式布局

| 屏幕尺寸 | 卡片布局 | 地图比例 |
|---------|---------|---------|
| xs (<576px) | 1 列 | 100% |
| sm (576-768px) | 2 列 | 100% |
| md (768-992px) | 2 列 | 66% |
| lg (≥992px) | 4 列 | 66% |

## 🗺️ 地图功能

### 两种显示模式

#### 模式一：高德地图模式（需要 API Key）

**配置**:
```env
NEXT_PUBLIC_AMAP_KEY=your-amap-key
```

**功能**:
- ✅ 真实地图显示
- ✅ 设备位置标注（彩色圆点）
- ✅ 点击显示信息窗体
- ✅ 自动缩放适配所有设备
- ✅ 悬停放大效果

**标记样式**:
```javascript
{
  width: 20px,
  height: 20px,
  background: status_color,
  border: '3px solid white',
  borderRadius: '50%',
  boxShadow: '0 2px 8px rgba(0,0,0,0.3)',
}
```

#### 模式二：简化地图模式（无 API Key）

**显示内容**:
- 设备卡片网格布局
- 状态指示灯（彩色圆点）
- 设备基本信息
- 坐标信息（如有）

**卡片内容**:
```

│ [待命]                  │
│ 📍 M3#泊位              │
│ 坐标：30.6721, 114.4669 │
└─────────────────────────┘
```

## 🔧 技术实现

### 文件结构

```
web/
├── app/
│   └── dashboard/
│       └── page.tsx          # 大屏主页面
├── components/
│   └── dashboard/
│       └── EquipmentMap.tsx  # 地图组件
└── .env.local                # 环境配置
```

### 关键代码

#### 1. 动态导入地图组件

```typescript
const EquipmentMap = dynamic(
  () => import('@/components/dashboard/EquipmentMap'),
  {
    ssr: false,
    loading: () => <div>地图加载中...</div>,
  }
);
```

#### 2. 地图模式判断

```typescript
const apiKey = process.env.NEXT_PUBLIC_AMAP_KEY;

if (!apiKey || apiKey === 'your-amap-key') {
  setUseSimpleMap(true); // 使用简化模式
} else {
  initAMap(apiKey); // 加载高德地图
}
```

#### 3. 设备标记创建

```typescript
equipments.forEach(equipment => {
  if (!equipment.latitude || !equipment.longitude) return;
  
  const marker = new AMap.Marker({
    position: [equipment.longitude, equipment.latitude],
    content: createMarkerContent(equipment.status),
  });
  
  marker.on('click', () => onEquipmentClick?.(equipment));
  marker.setMap(map);
});
```

## 🎯 使用说明

### 1. 配置高德地图 Key（可选）

访问 https://lbs.amap.com/ 申请 Key，然后在 `.env.local` 中配置：

```env
NEXT_PUBLIC_AMAP_KEY=your-amap-key
```

### 2. 启动服务

```bash
# 启动后端
cd /workspaces/SEM/eim
go run cmd/server/main.go

# 启动前端
cd /workspaces/SEM/eim/web
npm run dev
```

### 3. 访问大屏

浏览器访问：`http://localhost:3000/dashboard`

## ⚠️ 注意事项

### 1. 高德地图 Key

- 未配置 Key 时使用简化模式
- 简化模式显示设备卡片网格
- 建议申请 Key 以获得最佳体验

### 2. 坐标数据

- 32 台设备有完整坐标
- 6 台设备坐标为 (0, 0)
- 建议更新所有设备坐标

### 3. 性能优化

- 设备数量 < 100：实时刷新
- 设备数量 > 100：建议手动刷新
- 地图标记过多时考虑聚合
