## 已知问题

1. **扫码功能**
   - 部分浏览器可能需要 HTTPS 才能访问摄像头
   - iOS Safari 可能需要用户授权

2. **电子签名**
   - 签名图片未压缩，可能较大
   - 需要添加清空确认提示

3. **API 集成**
   - 部分 API 字段名称需要统一（snake_case vs camelCase）
   - 需要添加 loading 状态和错误处理
