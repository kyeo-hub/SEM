## 📝 开发日志

### 2026-03-11 - 大屏展示功能完善 ✅

**版本**: v2.4 | **开发者**: @kyeo-hub

#### 新增功能

1. **大屏展示页面 (/dashboard)**
   - 深色科技感主题设计
   - 统计卡片（设备总数、作业中、待命、故障）
   - 实时时间显示（每秒更新）
   - 自动数据刷新（每 10 秒）

2. **设备分布地图**
   - 高德地图集成（需要 API Key）
   - 简化地图模式（无 API Key 时显示卡片列表）
   - 设备标记（颜色区分状态）
   - 点击显示设备信息
   - 地图/列表切换开关

3. **设备列表展示**
   - 表格展示所有设备
   - 支持分页、排序
   - 详情弹窗（基本信息、状态、坐标）

4. **地图组件 (EquipmentMap)**
   - 动态加载高德地图
   - 自动缩放适配设备分布
   - 标记点交互（悬停放大、点击详情）
   - 降级处理（无 API Key 时显示卡片）

#### 技术实现

- **动态导入**: Next.js dynamic import (避免 SSR 问题)
- **实时刷新**: setInterval 定时轮询
- **状态管理**: React useState + useEffect
- **地图降级**: API Key 检测 + 简化模式 fallback
