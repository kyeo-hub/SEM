## 📌 注意事项

1. 使用 Next.js App Router 架构
2. 响应式设计，一套代码适配 PC 和移动端
3. 图表使用 ECharts 或 AntV
4. 地图使用高德地图 API
5. 所有表单需要实现前端验证
6. H5 移动端无需审核，开发完即可使用
