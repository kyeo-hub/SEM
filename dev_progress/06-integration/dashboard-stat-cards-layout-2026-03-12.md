# Dashboard 统计卡片布局优化

**日期**: 2026-03-12
**版本**: v2.5
**状态**: ✅ 已完成

---

## 需求描述

将 Dashboard 的 5 个统计卡片调整为一排显示，避免卡片过大占用过多空间。

## 解决方案

### 响应式布局调整

使用 Ant Design 的响应式 Grid 系统，在不同屏幕尺寸下自动调整卡片布局：

```typescript
<Row gutter={[16, 16]} justify="center">
  <Col xs={24} sm={12} md={8} lg={6} xl={4}>
    {/* 卡片内容 */}
  </Col>
  {/* 重复 5 次 */}
</Row>
```

使用 `justify="center"`让卡片在行中居中显示。

### 断点说明

| 断点 | 屏幕宽度 | 卡片数/行 | 说明 |
|------|----------|-----------|------|
| `xs` | < 576px | 1 | 手机竖屏 |
| `sm` | ≥ 576px | 2 | 手机横屏/小平板 |
| `md` | ≥ 768px | 3 | 平板 |
| `lg` | ≥ 992px | 4 | 小屏电脑 |
| `xl` | ≥ 1200px | 5 | 大屏电脑（一排显示） |

### 字体和图标大小调整

为了适应更紧凑的布局，同时保持视觉协调：

| 元素 | 原大小 | 新大小 |
|------|--------|--------|
| 标题字体 | 16px | 14px |
| 数字字体 | 36px | 28px |
| 图标 | 24px | 20px |
| 卡片内边距 | 24px | 16px |
| Row gutter | 24px | 16px |
